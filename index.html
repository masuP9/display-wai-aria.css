display-wai-aria.css

This CSS file displays WAI-ARIA Web Accessibility Initiative-Accessible Rich Internet Applications ( [role], [aria-*] )

How to use

Just drag the display WAI-ARIA button to your bookmarks bar, and you'll be able to use display-wai-aria.css on any page:

display WAI-ARIA If you want try, you can just click the button
